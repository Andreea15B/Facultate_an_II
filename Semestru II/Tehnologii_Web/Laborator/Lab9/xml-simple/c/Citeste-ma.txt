Acest director contine exemplificari de procesare XML simplificata oferita 
de biblioteca libxml (disponibila, uzual, in distributiile Linux actuale);
detalii la http://xmlsoft.org/

* procxml.c  - program C care proceseaza (parcurge) un document XML,
               folosind functiile xmlTextReader
* Makefile   - fisier folosit pentru compilarea programului prin "make"
* autori.xml - un document XML de test

Toate aceste fisiere au fost scrise de Sabin-Corneliu Buraga (2004).
